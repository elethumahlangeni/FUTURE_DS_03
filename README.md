## Dataset
- **Source:** UCI Bank Marketing Dataset (bank-additional-full.csv)
- **Records:** 41,188 customer contacts
- **Target:** `y` — term deposit subscription (yes/no)

## Deliverables
| File | Description |
|------|-------------|
| `Marketing_Funnel_Dashboard.html` | Full interactive enterprise dashboard |
| `dashboard_builder.py` | Python data preprocessing & aggregation |

## Dashboard Sections
1. Executive Overview (KPI cards + filters)
2. Marketing Funnel Analysis (visual funnel + dropoff)
3. Campaign Performance (monthly, duration, frequency)
4. Customer Segmentation (job, age, education, heatmap)
5. Conversion Drivers (feature importance)
6. Time Analysis (seasonality, trends)
7. Predictive Insights (scoring, ideal profile)
8. Recommendations (8 executive action items)

## Key Findings
- Overall conversion rate: **11.27%**
- Best channel: Cellular (14.74% vs 5.23% telephone)
- Best month: March (50.6% conversion)
- Top segment: Students (31.4%) and Retirees (25.2%)
- #1 driver: Call duration (correlation = 0.41)
- Prior success prospects: **65.1% conversion rate**
